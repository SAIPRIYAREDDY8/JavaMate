    ☕ Story of Java

    📌 Origin of Java

    • Developed in: 1991
    • Organization: Sun Microsystems
    • Lead Engineer: James Gosling
    • Team Members: Mike Sheridan, Patrick Naughton
    • Original Name: Oak

    📌 Why it was named “Java”

    • Original name “Oak” was already registered
    • Name changed to avoid legal issues
    • Inspired by Java coffee ☕
    • Coffee from Java Island (Indonesia)
    • Developers used to drink this coffee during work

    📌 Important Timeline

    • 1991 → Project started (called Oak)
    • 1995 → Renamed to Java
    • 1995 → Official release by Sun Microsystems
    • Later → Became popular for web & enterprise apps

    📌 Ownership

    • Java is now owned by Oracle Corporation
